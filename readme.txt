== Description ==

BuddyPress Avatar Popover let's you add a popover box when hovering on the group/member avatars and gives you more information at a glance.

== Installation ==
= To Install: =

1.  Download the plugin file
2.  Unzip the file into a folder on your hard drive
3.  Upload the `/bpap/` folder to the `/wp-content/plugins/` folder on your site
4.  Single-site BuddyPres go to Plugins menu and activate there.
5.  For Multisite visit Network Admin -> Plugins and Network Activate it there.

== Frequently Asked Questions ==

= What other configurations do I need =
No. This plugin needs no configuration.
